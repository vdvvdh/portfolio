Een webapplicatie met een minimalistisch design, geinspireerd op Persona 3 Reload. Ik heb me gefocust op een strakke interface en soepele prestaties.

HTML5
CSS
TypeScript

# Key Features
Navigatie: Een menu dat wisselt tussen de projecten en nog meer.

Design: Een functionele interface met een high contrast design.
